# JOGLFAddon-BlendScreens

JOGLFAddon to blend two screens

# Installation

```xml
<dependency>
    <groupId>com.github.dabasan</groupId>
    <artifactId>joglfaddon-blendscreens</artifactId>
    <version>1.2.0</version>
</dependency>
```
